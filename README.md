# skyway-stream-recorder

(SkyWay WebRTC Gateway)[https://github.com/skyway/skyway-webrtc-gateway] を利用して、ストリームを録画するツールです。

## セットアップ

下記コマンドで、SkyWay WebRTC Gatewayとffmpegをインストールします。

```shell
$ make install
```

## 音声・映像をリッスン

音声のリッスン。

```shell
$ make listen-audio
```

映像のリッスン。

```shell
$ make listen-video
```
